# Jeu de Devinettes — Java

## Description
Jeu console en Java où le joueur doit deviner un nombre aléatoire
entre 1 et 100 en 10 essais maximum.

## Structure du projet
- GuessingGame.java : mode clavier
- GuessingGameFile.java : mode fichier
- guesses.txt : fichier d'entrées

## Comment exécuter

### Mode clavier
```bash
javac GuessingGame.java
java GuessingGame
```

### Mode fichier
```bash
javac GuessingGameFile.java
java GuessingGameFile
```

## Règles du jeu
- Nombre aléatoire entre 1 et 100
- 10 essais maximum
- Indication trop grand / trop petit après chaque essai
